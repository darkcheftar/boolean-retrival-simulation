Boolean Retrival

The (standard) Boolean model of information retrieval (BIR) is a classical information retrieval (IR) model and, at the same time, the first and most-adopted one. It is used by many IR systems to this day. The BIR is based on Boolean logic and classical set theory in that both the documents to be searched and the user's query are conceived as sets of terms (a bag-of-words model). Retrieval is based on whether or not the documents contain the query terms.
In Boolean retrieval model, there are some logical operators we are using like AND, OR, NOT. Using these operators, we write a query using keywords with which we search or retrieve required documents. BY using this website, we can retrieve the needed documents from provided documents using keywords written in query format. Query can be written similarly like (abcdef AND ghijkl OR mnopqr NOT stuvwx). We need to upload the documents from which we need to retrieve our needed documents using drag or upload options. Using this query, our model in this website will retrieve the required documents and display.
